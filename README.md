# image-generate-sample
Sample of generating image by Ruby

---

## Getting Started
### Install gem
```
$ gem install gdk3
$ gem install atk
```
(confirmed with gdk3-3.3.2)

### Install font
Noto Sans CJK JP  
https://www.google.com/get/noto/#sans-jpan

## Generate Image
```
$ ruby generate.rb base-image-path add-image-path
```
